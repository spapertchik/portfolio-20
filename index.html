N.C.C. Responsive Web Redesign

Context:

Nature Conservancy Canada is a NPO. We were asked to redesign an existing website for the organization to improve the UI and eliminate difficulties users were experiencing while browsing the site.

Project Brief:

Our fundamental task was to evaluate the website and improve upon it in such a way that was helpful to the organization and users. It was up to us to define what those improvements needed to be.

My Role:

Research, Data synthesis, Information architecture, Wireframing, Layout design, Mock-up building.
